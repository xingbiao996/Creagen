  - [x] 子任务 1.2：确立全局 VI 规范（默认主题色为 `#66ccff`，并支持自定义切换），深度定制 Arco Design 的 Less 变量。封装全局样式文件与排版组件，严格确保中文排版应用「」和『』引号规范。

  - [x] 子任务 2.1：开发视频解析服务（集成或参考 `douyin_parse` 逻辑，获取实际视频地址、提取视频标题、简介、评论等）。

  - [x] 子任务 3.1：基于 Arco Design 规范，设计并实现精美的现代主页，包含推荐抖音精选创作者、热门视频、优秀「创作配方卡」的展示流，以及醒目的 URL 输入分析区域。
  - [x] 子任务 3.2：开发全局加载状态与动画（在解析和 AI 分析期间提供良好的用户反馈体验）。
  - [x] 子任务 3.3：设计并实现「创作配方卡」详情页，使用图表或精美的卡片排版展示 7 个维度的分析结果，并高亮展示 3-5 条行动建议。所有中文文案需确保引号合规。
  - [x] 子任务 3.4：开发历史配方卡列表页，支持按创作者或时间维度进行浏览。

- [x] 任务 4：外部辅助工具开发
